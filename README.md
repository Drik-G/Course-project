Student record management system/ course project: a simple java console application that tracks student information using both flat file storage and database.

Structure:

1. Main.java: Runs the system and JDBC integration
  
2. Manager.java: core operation for the text file I/O and SQL execution

3. Student.java: Defines encapsulation for student objects

4. GraduateStudent.java: Extends Student to manage advanced academics

5. student.txt: text file for recording operations
